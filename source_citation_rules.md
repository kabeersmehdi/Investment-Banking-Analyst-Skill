# Source Citation Rules — M&A Analyst Agent

## CORE PRINCIPLE
Every non-trivial factual claim in any output must be:
1. Sourced from a live web search OR flagged as embedded benchmark
2. Cited inline AND in a source block
3. Dated (when was it accessed / published)
4. Assessed for reliability (Tier 1 / 2 / 3)

---

# SIMPLIFIED CITATION APPROACH

## For Internal Work Product (Valuation Memos, Risk Reports)
- Inline: Just use [Source Name, Date] after the claim
- End of document: Full source list with URLs

## For Client-Facing Materials (CIM, Teaser)
- NO inline citations (clutters the document)
- Industry data presented as fact with source available on request
- Source list maintained internally, not in the CIM
- Add footnote: "Industry data sourced from [Source Names]. 
  Details available upon request."



---

## WHAT REQUIRES CITATION

### Always Cite
- ✅ Transaction multiples (SDE or EBITDA)
- ✅ Market size figures
- ✅ CAGR / growth rate claims
- ✅ Industry trend statements ("the industry is growing because...")
- ✅ Comparable transaction references
- ✅ Active buyer / acquirer identification
- ✅ Regulatory or legal claims
- ✅ Labor market or wage data
- ✅ Any claim with a specific number

### Never Requires Citation
- ❌ General M&A process descriptions
- ❌ Deal structure mechanics (SBA loan terms, etc.)
- ❌ Agent's own analytical conclusions
- ❌ Financial math performed on client-provided data
- ❌ Standard valuation methodology descriptions

## CITATION ICONS

Use these icons throughout all outputs:

| Icon | Meaning |
|------|---------|
| 🌐 | Live web data — sourced and cited |
| 📊 | Embedded benchmark — not live |
| ⚠️ | Warning / flag / caution |
| 📚 | Source block follows |
| ✅ | Confirmed / verified |
| ❓ | Unverified / needs confirmation |

---

## CONFIDENCE LEVELS

Attach confidence levels to valuation ranges:

| Level | Meaning | When to Use |
|-------|---------|-------------|
| HIGH | 2+ live sources confirm range | Multiple Tier 1 sources found |
| MEDIUM | 1 live source OR 2 embedded benchmarks | Limited data available |
| LOW | Embedded benchmark only, no live data | Search returned no results |

Example:
> Valuation Range: $3.2M–$4.1M [CONFIDENCE: MEDIUM]
> Based on 1 live source (BizBuySell Q4 2024) and embedded
> benchmark data. Recommend verification before LOI.

---

## ANTI-HALLUCINATION RULES

1. NEVER invent a URL, publication, or data point
2. NEVER cite a source you did not actually retrieve
3. If unsure whether a source exists, search first
4. If search fails, use embedded benchmark and flag clearly
5. NEVER present a range as "current" without a live source
6. If a URL cannot be verified, write: [URL unverified — search manually]
7. Always prefer saying "I don't have live data on this" over fabricating
