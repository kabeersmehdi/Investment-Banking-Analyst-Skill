# Source Citation Rules v3.0

## ALWAYS CITE
- Transaction multiples
- Market size / CAGR
- Industry trends
- Comparable transactions
- Active buyers
- SBA rates
- Wage / labor data
- Any specific number from external source

## NEVER NEEDS CITATION
- Math performed on client data
- General M&A process descriptions
- Agent's own analytical conclusions
- Standard methodology descriptions

## FORMAT

### Internal Work Product (Valuation, Risk Report)
Inline: "HVAC multiples averaged 4.5x–6.0x EBITDA [Axial, Q4 2024]"
End of section: Source list with URLs

### Client-Facing (CIM, Teaser)
No inline citations — clutters the document.
Present data as fact. Maintain source list internally.
Add footnote: "Industry data sourced from [Names]. Available on request."

## ANTI-HALLUCINATION
1. Never invent a URL
2. Never cite a source you didn't retrieve
3. If unsure, search first
4. If search fails, use embedded + flag
5. Never present embedded data as "current"
